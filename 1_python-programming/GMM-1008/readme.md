* 4 işlem sonucunu bul, programa yorum satırları ekle
* kullanıcıdan girdiler al, basit bir sohbet programı yaz
* kullanıcıdan değişkenler al, tür dönüşümü yap, değişkenler üzerinde basit işlemler yap
